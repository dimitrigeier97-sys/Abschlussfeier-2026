# Abschlussfeier 2026

Statische Webseite fuer die Abschlussfeier 2026 mit OneDrive-Fotolink und Gluecksrad.

## OneDrive-Link

Der Button "Fotos in der Cloud" verweist auf den freigegebenen OneDrive-/SharePoint-Ordner der Abschlussfotos.

## Lokal testen

Die Seite besteht nur aus HTML, CSS und JavaScript. Sie kann direkt geoeffnet oder ueber einen kleinen lokalen Server getestet werden.

## Vercel-Veröffentlichung

1. Repository bei GitHub erstellen.
2. Diese Dateien hochladen oder per Git committen und pushen.
3. In Vercel ein neues Projekt aus dem GitHub-Repository importieren.
4. Framework Preset auf `Other` lassen.
5. Build Command leer lassen.
6. Output Directory leer lassen oder `.` verwenden.
7. Deploy klicken.

Vercel ist fuer viele gleichzeitige Besucher gut geeignet, weil diese statische Seite ohne Serverlogik ausgeliefert wird.
